# Quick Start

Get your AI agent running on Starknet in **under 5 minutes**. Choose your path:

| Path | Best For |
|------|----------|
| **[Scaffold a new agent](#option-a-scaffold-a-new-agent)** | Starting a new project from scratch |
| **[Clone the monorepo](#option-b-clone-the-monorepo)** | Contributing or exploring examples |

## Prerequisites

Before you begin, make sure you have:

- **Node.js 18+** installed ([download](https://nodejs.org/))
- **pnpm** package manager (`npm install -g pnpm`)

## Option A: Scaffold a New Agent

The fastest way to start is with `create-starknet-agent`:

```bash
npx create-starknet-agent@latest my-agent
```

This launches an interactive wizard that:
1. Asks for your project name
2. Lets you choose a template (minimal, defi, or full)
3. Configures your network (Sepolia testnet by default)
4. Installs dependencies automatically

<Callout type="tip" title="Quick Setup">
  Use `npx create-starknet-agent@latest my-agent -y` to accept all defaults and skip prompts.
</Callout>

### Templates

| Template | Description |
|----------|-------------|
| **minimal** | Wallet + basic operations (balance, transfer) |
| **defi** | Wallet + AVNU swaps and DeFi monitoring |
| **full** | All features + ERC-8004 identity + A2A readiness |

After scaffolding:

```bash
cd my-agent
cp .env.example .env
# Edit .env with your wallet credentials
pnpm start
```

## Configure Your Environment

Edit your `.env` file with Starknet credentials:

```env
# Starknet RPC endpoint (get a free key from Alchemy or Infura)
STARKNET_RPC_URL=https://starknet-sepolia.g.alchemy.com/v2/YOUR_KEY

# Your Starknet account address
STARKNET_ACCOUNT_ADDRESS=0x...

# Your account private key (NEVER share or commit this!)
STARKNET_PRIVATE_KEY=0x...
```

<Callout type="warning" title="Security Warning">
  Never commit your private key to version control. The `.env` file is already in `.gitignore`, but double-check before pushing.
</Callout>

### Getting Your Credentials

You'll need both a wallet and an RPC endpoint:

**1. Ready Wallet**

1. Install [Ready Wallet browser extension](https://www.ready.co/download-ready-wallet)
2. Create a new wallet or import existing to **Sepolia Testnet**
3. Export private key: Settings → Account → Export Private Key
4. Copy your account address from the wallet
5. Get testnet tokens from the [Starknet Faucet](https://starknet-faucet.vercel.app/)

**2. Get a Free RPC Endpoint**

- [Alchemy](https://www.alchemy.com/) - Create account → New App → Starknet Sepolia
- [Infura](https://infura.io/) - Create account → New Project → Starknet
- [Other providers](https://www.starknet.io/fullnodes-rpc-services/) - Explore more RPC services

<Callout type="tip" title="New to Starknet?">
  We recommend [Ready Wallet](https://www.ready.co/download-ready-wallet) for beginners. Other popular options include [Braavos](https://braavos.app/) (advanced security features), [Bitget Wallet](https://web3.bitget.com/en/assets/starknet-wallet) (multi-chain), and [OKX Wallet](https://www.okx.com/web3) (mobile-first).
</Callout>

## Connect to an AI Assistant

The MCP server works with any AI assistant that supports the Model Context Protocol.

<Callout type="info" title="Using Other AI Assistants?">
  This quick start shows Claude Desktop setup, but the MCP server works with ChatGPT, Codex, and other MCP-compatible assistants. See the [full MCP Server guide](/docs/guides/mcp-server) for setup instructions for different platforms.
</Callout>

### Claude Desktop Setup

Add this to your Claude Desktop config (`~/Library/Application Support/Claude/claude_desktop_config.json` on macOS):

```json
{
  "mcpServers": {
    "starknet": {
      "command": "node",
      "args": ["/path/to/starknet-agentic/packages/starknet-mcp-server/dist/index.js"],
      "env": {
        "STARKNET_RPC_URL": "https://starknet-sepolia.g.alchemy.com/v2/YOUR_KEY",
        "STARKNET_ACCOUNT_ADDRESS": "0x...",
        "STARKNET_PRIVATE_KEY": "0x..."
      }
    }
  }
}
```

### Test the Connection

Restart Claude, then try the following prompt:

> Check my STRK balance

Claude will use the `starknet_get_balance` tool to fetch your balance.

## Troubleshooting

**"Cannot find module" error**

Make sure you ran `pnpm build` after installation.

**"Invalid account address" error**

Check that your `STARKNET_ACCOUNT_ADDRESS` is a valid Starknet address (starts with `0x`, 64 hex characters).

**"Insufficient balance" error**

Get testnet tokens from the [Starknet Faucet](https://starknet-faucet.vercel.app/).

**RPC connection timeout**

Verify your `STARKNET_RPC_URL` is correct and your API key is valid.
